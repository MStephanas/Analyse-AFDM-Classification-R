# Analyse-AFDM-Classification-R
Projet de segmentation de données mixtes via AFDM et CAH sous R
Introduction :  L'objectif général est de réaliser une analyse comparative des types de jeux de hasard pratiqués en ligne et en points physiques par les étudiants de l'Université de Parakou. De manière spécifique, il s’agit de décrire ces pratiques selon le mode d'accès, d'analyser les associations entre les deux formes de jeu, et d'établir une typologie des étudiants joueurs.

Méthodologie :  L'AFCM a été appliquée à un tableau disjonctif complet croisant les individus (étudiants) et les modalités des questions Q2406 et Q2407. 

Résultats :  À partir des coordonnées factorielles des individus issues de l'AFCM, une classification ascendante hiérarchique a été réalisée en utilisant :
•	La distance euclidienne comme mesure de dissimilarité ;
•	La méthode de Ward comme critère d'agrégation.
Le nombre optimal de classes a été déterminé à l'aide du critère du saut d'inertie et de l'examen du dendrogramme.
Chaque classe obtenue a ensuite été caractérisée :
•	Par ses pratiques de jeux dominantes (modalités sur-représentées) ;
•	Par son profil sociodémographique (tests du Khi² pour identifier les variables associées).
